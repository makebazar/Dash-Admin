# Интерфейс Сотрудника (Employee Portal)

Портал сотрудника оптимизирован для повседневной работы админа клуба. Основной акцент сделан на операционную деятельность и выполнение регламентов.

## 1. Личный дашборд
**Путь:** `/employee/dashboard`
Экран выбора клуба и управления личным профилем. Здесь сотрудник видит список клубов, в которых он трудоустроен, и свою роль.

## 2. Дашборд клуба (Рабочее место)
**Путь:** `/employee/clubs/[clubId]`
Основной рабочий экран после выбора клуба.

### Ключевые возможности:
- **Управление сменой (Wizards):**
    - **Открытие смены:** Пошаговый процесс инвентаризации остатков, сверки кассы.
    - **Закрытие смены:** Сдача остатков, отчет по выручке, передача комментариев следующей смене.
- **KPI и Зарплата:** Моментальное отображение заработанных средств за текущую неделю/месяц, включая бонусы за KPI.
- **Интерактивные виджеты:**
    - Контроль экранов (Signage) напрямую из портала.
    - Управление промо-акциями (выпуск билетов, активация промокодов).

## 3. Кассовый терминал (POS)
**Путь:** `/employee/clubs/[clubId]/pos`
Специализированный интерфейс для продажи товаров и услуг.

### Функционал:
- Быстрый выбор товаров по категориям.
- Поддержка сканера штрих-кодов.
- Интеграция с системой лояльности (продажа игроку с учетом его скидок).
- Блокировка кассы, если не выполнена входная инвентаризация.

## 4. Задачи и Регламенты
**Путь:** `/employee/clubs/[clubId]/tasks`
Список всех дел на смену.

### Типы задач:
- **Чек-листы:** Ежедневная проверка чистоты, работоспособности оборудования. Часто требуют фото-фиксации результата.
- **Обслуживание (Maintenance):** Специальные технические сессии по чистке ПК или периферии.
- **Поручения:** Разовые задачи от управляющего.

## 5. База знаний (Knowledge Base)
**Путь:** `/employee/clubs/[clubId]/kb`
Централизованное хранилище инструкций: от скриптов продаж до гайдов по устранению типовых поломок.
