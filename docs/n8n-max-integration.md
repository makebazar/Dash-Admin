# Интеграция DashAdmin с мессенджером Max через n8n

Этот документ описывает, как связать AI-агента, работающего на базе n8n, с API нашего проекта для получения аналитических данных и отправки их в чат мессенджера Max.

## 1. Ключевые компоненты

- **DashAdmin Reporting API:** Наш внутренний API, который предоставляет аналитику. 
  - **Адрес:** `/api/clubs/{clubId}/reporting`
  - **Защита:** Требует API-ключ в заголовке `Authorization: Bearer <DASHADMIN_API_KEY>`.

- **Мессенджер Max:** Платформа для общения с пользователем.
  - **API:** `https://platform-api.max.ru`
  - **Аутентификация:** Требует токен бота в заголовке `Authorization: <MAX_BOT_TOKEN>`.

- **n8n:** Платформа-посредник, которая автоматизирует процесс.
  - **Нода:** Используем `n8n-nodes-max` (community node).

## 2. Схема работы (Workflow)

1.  **Пользователь пишет боту:** Пользователь отправляет сообщение боту в мессенджере Max, например: "Какая выручка за вчера?"

2.  **Срабатывает триггер в n8n:** Установленная нода `n8n-nodes-max` настроена как триггер. Она получает это сообщение от API Max через Webhook.

3.  **Запрос к AI-модели:**
    - Текст сообщения ("Какая выручка за вчера?") отправляется в большую языковую модель (LLM, например, GPT-4).
    - Вместе с текстом модели передается "инструкция" — наш `openapi.json` файл, описывающий возможности DashAdmin Reporting API.

4.  **Function Calling (Магия AI):**
    - Модель анализирует вопрос и понимает, что для ответа ей нужны данные.
    - Она "смотрит" в `openapi.json` и находит подходящий инструмент — `getReport`.
    - Модель извлекает из вопроса параметры: `query: 'revenue'` и `period: 'yesterday'`.
    - AI-модель **не отвечает сама**, а возвращает в n8n команду: "Вызови функцию `getReport` с такими-то параметрами".

5.  **n8n делает запрос к нашему API:**
    - n8n получает команду от AI.
    - Он формирует и выполняет HTTP-запрос к нашему API:
      `GET https://dashadmin.yourdomain.com/api/clubs/1/reporting?query=revenue&period=yesterday`
    - В заголовок запроса n8n подставляет наш секретный ключ: `Authorization: Bearer da-report-sk_...`.

6.  **Наш API отвечает:** DashAdmin API проверяет ключ, выполняет запрос к базе данных и возвращает в n8n JSON с реальными цифрами: `{"total_revenue": 12500, ...}`.

7.  **Финальный ответ AI:**
    - n8n отправляет полученные от API данные обратно в AI-модель с просьбой: "Сформируй ответ для пользователя на основе этих данных".
    - Модель генерирует человекопонятный ответ: "Выручка за вчерашний день составила 12 500."

8.  **n8n отправляет ответ в Max:**
    - n8n использует ноду `n8n-nodes-max` (действие "Send Message").
    - Он отправляет сгенерированный текст ответа пользователю в чат Max.

Таким образом, мы получаем полностью автоматизированную систему, где AI выступает в роли "мозгового центра", который умеет пользоваться созданными нами инструментами (API) для ответа на вопросы.
